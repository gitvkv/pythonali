# The os Module for System Interactions

    ---
    
    ### 🌱 Context Introduction
    
    Python's **os module** is a built-in toolkit that allows your scripts to interact directly with the operating system. Whether you need to navigate folders, check file paths, read environment variables, or run system commands, the **os module** gives you the power to control the underlying system from within your Python code. It acts as a bridge between your script and the operating system's core functions.
    
    ---
    
    ### ⚙️ What is the os Module?
    
    The **os module** provides a portable way of using operating system-dependent functionality. It includes functions for:
    
    - **File and directory operations** — creating, removing, renaming, and navigating folders
    - **Path manipulations** — joining, splitting, and checking paths
    - **Environment variables** — reading and setting system-level variables
    - **Process management** — getting current process ID, user information
    - **Running system commands** — executing shell commands from Python
    
    ---
    
    ### 🛠️ Common os Module Functions
    
    Here are the most frequently used functions for system interactions:
    
    **Working with Current Directory**
    
    - **os.getcwd()** — Returns the current working directory as a string
    - **os.chdir(path)** — Changes the current working directory to the specified path
    
    **Listing and Creating Directories**
    
    - **os.listdir(path)** — Returns a list of all files and folders in the given directory
    - **os.mkdir(path)** — Creates a single new directory
    - **os.makedirs(path)** — Creates a directory and any necessary parent directories
    
    **Removing Files and Directories**
    
    - **os.remove(path)** — Deletes a single file
    - **os.rmdir(path)** — Removes an empty directory
    - **os.removedirs(path)** — Removes a directory and its empty parent directories
    
    **Checking Paths**
    
    - **os.path.exists(path)** — Returns **True** if the path exists, otherwise **False**
    - **os.path.isfile(path)** — Returns **True** if the path is a file
    - **os.path.isdir(path)** — Returns **True** if the path is a directory
    
    **Environment Variables**
    
    - **os.environ** — A dictionary-like object containing all environment variables
    - **os.getenv('VAR_NAME')** — Returns the value of a specific environment variable
    - **os.environ['VAR_NAME'] = 'value'** — Sets a new environment variable
    
    **System Information**
    
    - **os.name** — Returns the operating system name (posix, nt, java)
    - **os.getlogin()** — Returns the name of the user logged in
    - **os.getpid()** — Returns the current process ID
    
    ---
    
    ### 📊 Comparison: os.path vs os Module Functions
    
    | Feature | os.path Functions | os Module Functions |
    |---------|------------------|-------------------|
    | **Purpose** | Path manipulation and checks | Direct system operations |
    | **Join paths** | **os.path.join()** | Not available |
    | **Split paths** | **os.path.split()** | Not available |
    | **Check existence** | **os.path.exists()** | Not available |
    | **Create directory** | Not available | **os.mkdir()** |
    | **List directory** | Not available | **os.listdir()** |
    | **Change directory** | Not available | **os.chdir()** |
    | **Get file size** | **os.path.getsize()** | Not available |
    
    ---
    
    ### 🕵️ Practical Examples for Engineers
    
    **Example 1: Navigating and Listing Directories**
    
    Start by importing the module: **import os**
    
    Get your current location: **current_dir = os.getcwd()** — this stores the path as a string
    
    List everything in that directory: **items = os.listdir(current_dir)** — this gives you a list of file and folder names
    
    Change to a different folder: **os.chdir('/tmp')** — now your script is working from the /tmp directory
    
    **Example 2: Creating a Project Folder Structure**
    
    Define a base path: **base_path = '/home/user/projects'**
    
    Create nested folders: **os.makedirs(base_path + '/logs/2024')** — this creates both the logs folder and the 2024 subfolder in one command
    
    Check if it worked: **os.path.exists(base_path + '/logs/2024')** — returns **True** if successful
    
    **Example 3: Reading Environment Variables**
    
    Get a specific variable: **home_dir = os.getenv('HOME')** — returns something like **/home/username**
    
    List all environment variables: **for key, value in os.environ.items(): print(f'{key} = {value}')** — this prints every variable and its value
    
    Set a temporary variable: **os.environ['MY_VAR'] = 'hello'** — this variable exists only while your script runs
    
    **Example 4: Checking File Types**
    
    Define a path: **path_to_check = '/etc/hosts'**
    
    Check what it is: **if os.path.isfile(path_to_check): print('It is a file')** — returns **True** for /etc/hosts
    
    Check if it exists: **if os.path.exists(path_to_check): print('Path exists')** — always good to check before acting
    
    ---
    
    ### 🧠 Key Takeaways for Engineers
    
    - The **os module** is your direct line to the operating system from Python
    - Always import it first: **import os**
    - Use **os.path** functions for safe path manipulations across different operating systems
    - Environment variables accessed via **os.environ** are case-sensitive
    - Directory creation with **os.makedirs** is safer than **os.mkdir** because it creates parent folders automatically
    - Always check if a path exists before trying to read or write to it
    - The **os module** works on Windows, macOS, and Linux, making your scripts portable

    The `os` module provides functions to interact with your computer's operating system, allowing you to work with files, folders, and system information from within Python.
    
    ---
    
    ### 🖥️ Example 1: Getting the current working directory
    
    Shows how to find out which folder your Python script is currently running in.
    
    ```python
    import os
    
    current_folder = os.getcwd()
    print(current_folder)
    ```
    
    📤 Output: **/home/user/projects**
    
    ---
    
    ### 📂 Example 2: Listing files and folders in a directory
    
    Shows how to see all items inside a specific folder.
    
    ```python
    import os
    
    items = os.listdir(".")
    print(items)
    ```
    
    📤 Output: **['main.py', 'data.csv', 'images', 'notes.txt']**
    
    ---
    
    ### 🗂️ Example 3: Creating a new folder
    
    Shows how to make a new directory for organizing files.
    
    ```python
    import os
    
    os.mkdir("new_project_folder")
    print("Folder created successfully")
    ```
    
    📤 Output: **Folder created successfully**
    
    ---
    
    ### 🔄 Example 4: Renaming a file
    
    Shows how to change the name of an existing file.
    
    ```python
    import os
    
    os.rename("old_report.txt", "new_report.txt")
    print("File renamed")
    ```
    
    📤 Output: **File renamed**
    
    ---
    
    ### 🧹 Example 5: Checking if a file exists before deleting
    
    Shows a safe way to remove a file only if it is present.
    
    ```python
    import os
    
    file_name = "temp_data.txt"
    
    if os.path.exists(file_name):
        os.remove(file_name)
        print("File deleted")
    else:
        print("File not found")
    ```
    
    📤 Output: **File deleted**

    | `os.rename()` | Renames a file or folder | Update file names |
    | `os.remove()` | Deletes a file | Clean up old files |
